Minimal AI Agent Playground

This project demonstrates how a simple Flask app becomes a minimal AI agent. The system keeps the LLM’s general intelligence for normal prompts, but it can also trigger a small built-in tool when you ask it to look up a key.

General Intelligence

Ask normal questions like “What is Flask?” and the model responds like a regular chatbot.

Tool Use

Use prompts like lookup flask to trigger the app’s dictionary tool.

Agentic Behavior

The app detects intent, calls a function, and brings the result back into the final answer.

Before you test

This is a local AI system using Flask and a free LLM setup. Responses are generated live, not pre-written.
Please wait after sending a message. The system may be a bit slow, especially on the first prompt or on lighter hardware. Give it up to a minute before the response appears.
