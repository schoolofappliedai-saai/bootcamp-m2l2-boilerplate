SAAI Project • Design → Develop → Deploy → Defend

Build a Minimal AI Agent, not just a chatbot.

This project teaches students how to use Flask to create a simple web interface, connect it to an LLM through Ollama, and add one minimal tool in Python. The result is a working beginner-friendly AI agent that keeps the model’s general intelligence while also performing a specific action when prompted correctly.

What this project is

A one-page Flask chatbot app with local LLM integration and a dictionary lookup tool.

What students learn

How tool use makes an app agentic: user input, intent detection, function call, and natural response.

Why it matters

It is the first practical step from basic web development into applied AI product engineering.
